TrailOps Dashboard (React) - Phase 0 Scaffold

This is a minimal React+Vite app that proves the FastAPI plumbing works.

Run (from C:\trail_ops):

1) Start API
   python -m app.tools.run_api

2) Start UI
   cd frontend
   npm install
   npm run dev

Then open:
- http://127.0.0.1:5173

Streamlit will iframe this at:
- Dashboard page (app/pages/dashboard.py)
